About us

Hospital-Acquired Pressure Injury Risk Prediction System for Elderly Patients

System Introduction

It is a risk prediction system for hospital-acquired pressure injuries (HAPI) in elderly patients based on machine learning technology.

The core of this system uses advanced predictive algorithms to quickly build high-precision risk assessment models by comprehensively analyzing patients' clinical data, helping medical staff identify high-risk patients early and enabling timely intervention.

The system adopts a multi-model ensemble strategy, integrating various machine learning algorithms, with high accuracy, high sensitivity, and high specificity. It also supports real-time processing, data visualization, result interpretation, and batch prediction functions, providing a scientific basis for clinical decision-making.

Main functions of the system

HAPI Risk Prediction

  • Single prediction: Input data from a single patient for individualized risk assessment
  • Batch Prediction: Process data from multiple patients in batches and generate comprehensive risk reports
  • Multi-model Ensemble: Combining XGBoost, Random Forest, and other models to improve prediction accuracy
  • Risk Level Assessment: Automatically classify patient risk levels and provide intervention recommendations

Medical Intelligent Assistance

  • Intelligent Consultation: An Intelligent Q&A System Based on Natural Language Processing
  • Data Dashboard: Visualization of System Operating Status and Forecast Data Statistics
  • Feature Importance Analysis: Identifying Key Factors Affecting Risk
  • Prediction result explanation: Provide visual analysis of the model's decision basis

System Management

  • User Management: Multi-level User Permission Management
  • Data Security: Encrypted Transmission and Anonymization Ensure Patient Privacy
  • Multi-theme support: includes day mode and night mode
  • Consultation System: User Inquiries and Professional Doctor Responses

Medical Experts Q&A

  • DeepSeek Medical Expert Based on LoRA Fine-Tuning
  • Real-time Q&A interactive interface
  • Supports medical consultations, diagnostic advice, and other functions
  • Supports voice input and voice reading, providing a natural communication experience

Data Dashboard

  • System usage statistics visualization
  • Prediction and Q&A Trend Chart
  • Query Historical Records of Prediction Tasks
  • User Activity Monitoring and Analysis

Technology stack

Backend technology

  • Framework: FastAPI Framework - Asynchronous Support, Automatic API Documentation, High-Performance Routing
  • Machine Learning: Ensemble models such as XGBoost and Random Forest - used for HAPI risk prediction
  • Database: SQLAlchemy PostgreSQL - Relational data storage, transaction support, data migration
  • Data Processing: Pandas, NumPy - Data Preprocessing, Feature Engineering

Frontend technology

  • UI Framework: Bootstrap 5 - Responsive Design, Theme Customization, Component Library
  • Real-time communication: WebSocket - Real-time data updates, bidirectional communication, reconnection after disconnection
  • Chart: Chart.js - Data Visualization, Real-Time Updates, Interactive Charts
  • Interaction: jQuery - DOM Manipulation, Event Handling, AJAX Requests

Medical risk prediction tool

  • Predictive Model: Integrate multiple machine learning algorithms
  • Feature Analysis: Feature Importance Evaluation Tool
  • Result Explanation: Model Explanation Framework

System platform

  • Containerization: Docker - Environment isolation, rapid deployment, horizontal scaling
  • Web Server: Nginx - Reverse Proxy, Load Balancing, Static Resource Service
  • Process Management: Gunicorn - Multiple worker processes, graceful restart, monitoring interface
  • Security Verification: JWT - Token Authentication and Access Control
